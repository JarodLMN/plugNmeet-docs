# Delete recording

End point: `/recording/delete`

| Field     | Type   | Position | Required | Description                          |
| --------- | ------ | -------- | :------- | ------------------------------------ |
| record_id | string | root     | Yes      | Recording Id that you want to delete |

**Example**:

```
{
  "record_id": "RM_RKD4jeiFMZDS-1645753430902"
}
```
